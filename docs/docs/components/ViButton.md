# ViButton

Displays a button or a component that looks like a button.

## Demo

::: tip Primary

<p></p>
<ViButton style="padding: 10px 25px">Button</ViButton>
<p></p>
:::

## Usage

```ts
import { ViButton } from 'astral-vui'
```

```vue
<ViButton>Button</ViButton>
```
